# Promo Banner

The "promo banner" component is used to provide for promotion / advertisement banners uploaded via the SuperSport CMS.

These banners are traditionally included within articles as embed using the CMS to select from the promotional banners data repository, and are inserted using the Rich Text Editor (RTE) into the article body content.
