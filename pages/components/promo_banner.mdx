# Promo Banner

The "promo banner" component is used to provide for promotion / advertisement banners uploaded via the SuperSport CMS.

These banners are traditionally included within articles as embed using the CMS to select from the promotional banners data repository, and are inserted using the Rich Text Editor (RTE) into the article body content.

## Component Styling

The SuperSport Web Component library follows the Shadow DOM specification in order to encapsulate styles and markup. To override default styling of components [shadow parts](#shadow-parts) are used and allow developers to style inside a shadow tree, from outside of that shadow tree.

### Styling all instances

Given the below example, two instances of the promo banner component are rendered.

```css copy
<ss-promo-banner banner-id="fae0a53e-99aa-4bf4-9950-142fa9110607"></ss-promo-banner>
<ss-promo-banner banner-id="7eccb58e-da70-4957-8f85-afeedfe43816"></ss-promo-banner>
```

To style all instances of the component, the following CSS is used to target the component by name, including its associated parts.

```css copy
<style>
  ss-promo-banner::part(banner) {
    display: flex;
    justify-content: center;
    align-items: center;
  }

  ss-promo-banner::part(image) {
    background-color: red;
    padding: 50px;
  }
</style>
```

### Styling a single instance

Given the below example, two instances of the promo banner component are rendered. One contains an id attribute which is used to target the component.

```html copy
<ss-promo-banner
	banner-id="fae0a53e-99aa-4bf4-9950-142fa9110607"
></ss-promo-banner>
<ss-promo-banner
	id="styled-banner"
	banner-id="7eccb58e-da70-4957-8f85-afeedfe43816"
></ss-promo-banner>
```

To style a single instances of a component, the following CSS is used to target the component by CSS id, including its associated parts.

```css copy
<style>
  #styled-banner::part(banner) {
    display: flex;
    justify-content: center;
    align-items: center;
  }

  #styled-banner::part(image) {
    background-color: red;
    padding: 50px;
  }
</style>
```

## Usage

### How to Use `ss-promo-banner`

This component is used to provide a way to greet a user using their first, middle, and last name.
This component will properly format the provided name, even when all fields aren't provided:

```html copy
<ss-promo-banner first="Stencil"></ss-promo-banner>
<ss-promo-banner first="Stencil" last="JS"></ss-promo-banner>
```
